# jig.jpインターン課題
React.js+Redux+TypeScript+Material-UI

# 実装した機能
 * json中身一覧表示
 * 10件、30件、全件の指定
 * category, city, event_placeでの絞り込み
 * リアルタイムキーワード検索

絞り込み要件についてはcategory, city, event_placeをそれぞれ変更すると他の要素の候補もそれに合わせ変更し、無効な組合せを選ぶことが無いようにしました。

# ディレクトリ構成
``npm run build```でdist以下にWebページが出力されます。
